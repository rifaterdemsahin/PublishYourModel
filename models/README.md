# Data Model Schema

This directory contains the data model schema files for the project.

## Files

- `schema.sql` - SQL schema definition for the database tables

## Schema Overview

The data model includes the following tables:
- **users** - User information
- **products** - Product catalog
- **orders** - Order header information
- **order_items** - Order line items

## Usage

To create the database schema, run:
```sql
-- Execute schema.sql in your database
```
