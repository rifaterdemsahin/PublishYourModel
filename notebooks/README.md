# Notebook Examples

This directory contains Jupyter notebook examples for data analysis and exploration.

## Notebooks

- `example_analysis.ipynb` - Example notebook demonstrating basic data analysis and ETL operations

## Running Notebooks

After running `setup.sh`, you can launch Jupyter Lab:

```bash
jupyter lab --port 8888
```

Or use Jupyter Notebook:

```bash
jupyter notebook
```

## Requirements

- jupyter
- jupyterlab
- pandas
- numpy
- matplotlib
